# Team CLI Project Guide

This file mirrors the guide generated by `team init`.

Use it as the short project-local reference for teams that adopt `codemate`.

## Workflow

```text
plan -> implement -> review -> test
```

## Common Commands

```bash
team doctor
team run "Describe the task"
team status
team logs --step implement
team diff
team accept --commit --message "Describe accepted changes"
team reset
```

## Artifacts

Run artifacts are stored under:

```text
.team/runs/<run-id>/
```

Important files:

- `task.md`
- `state.json`
- `<step>.prompt.md`
- `<step>.output.md`
- `<step>.raw.log`
- `<step>.diff`
- `test.log`
- `final.md`
