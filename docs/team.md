# Team CLI Project Guide

This file mirrors the guide generated by `codemate init`.

Use it as the short project-local reference for teams that adopt `codemate`.

## Workflow

```text
plan -> implement -> review -> test
```

## Common Commands

Start an interactive session (the usual way to work):

```bash
codemate            # then type tasks; /help lists slash commands
```

Or run a single task non-interactively:

```bash
codemate doctor
codemate run "Describe the task"
codemate status
codemate logs --step implement
codemate diff
codemate accept --commit --message "Describe accepted changes"
codemate reset
```

## Artifacts

Run artifacts are stored under:

```text
.team/runs/<run-id>/
```

Important files:

- `task.md`
- `state.json`
- `<step>.prompt.md`
- `<step>.output.md`
- `<step>.raw.log`
- `<step>.diff`
- `test.log`
- `final.md`
